- **서비스 등록 여부**: 악성코드는 새로운 서비스를 등록하여 시스템 시작 시 함께 실행되도록 설정하였습니다. 이는 사용자가 눈치채기 어려운 방식으로 악성코드를 실행하기 위한 방법 중 하나입니다.

② 근거 제공
- Command and Scripting Interpreter (T1059): 악성코드가 PowerShell 및 다른 스크립팅 인터프리터를 사용하여 명령을 실행한 흔적이 발견되었습니다. 로그 파일에서 "powershell.exe"가 실행되었으며, 의심스러운 스크립트가 실행된 기록이 확인되었습니다.
- Obfuscated Files or Information (T1027): 샘플 파일 내에 난독화된 코드가 다수 발견되었습니다. 분석 도중, Base64 및 XOR로 인코딩된 문자열이 디코딩되어 악성 행위를 수행하는 부분이 밝혀졌습니다.
- Process Discovery (T1057): 시스템에서 실행 중인 프로세스 목록을 수집하는 명령어가 발견되었습니다. 로그에는 "tasklist" 명령어가 실행된 기록이 남아 있습니다.
- Ingress Tool Transfer (T1105): 원격 서버로부터 추가 악성 페이로드를 다운로드하는 행위가 탐지되었습니다. 네트워크 트래픽 분석 결과, 특정 IP 주소로 연결하여 파일을 다운로드한 흔적이 있습니다.
- System Information Discovery (T1082): 시스템 정보를 수집하는 코드가 포함되어 있습니다. "systeminfo" 명령어가 실행되어 OS 및 하드웨어 정보가 수집된 로그가 발견되었습니다.
- System Services (T1569): 자동 실행을 위해 시스템 서비스를 생성하거나 수정한 증거가 발견되었습니다. "sc.exe"를 통해 서비스가 등록된 로그가 존재합니다.

1. 초기 실행 단계에서 악성코드는 PowerShell 스크립트를 통해 난독화된 코드를 로드하고 실행합니다.
2. 시스템 정보 및 프로세스 정보를 수집하여 공격자가 원하는 정보를 서버로 전송합니다.
3. 추가 페이로드를 다운로드하기 위해 원격 서버와 통신을 시도하며, 다운로드된 파일은 추가 악성 행위를 수행합니다.
4. 지속성을 확보하기 위해 시스템 서비스를 생성 및 수정하여, 시스템 재부팅 시에도 자동으로 실행되도록 설정합니다.
